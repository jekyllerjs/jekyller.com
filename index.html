Serverless CMS for your Static Web Site

Optional

Jekyller services are an optional add on to a Jekyll site so your site keeps working even when the services are not available. Built with node.js using Serverless and hosted on AWS lambda.

Simple

All of your main site is built with Jekyll, an easy way to setup a website without having to install databases or web servers. Jekyller adds external services that work with the site github repo to maintain the content.

Free

Jekyller services are 100% open source and free to use in any way you desire. No hosting fees. No restrictions. Setup your own AWS account and quickly deploy to get started. Works great using AWS free tier account.
